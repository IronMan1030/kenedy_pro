How to update the version name and number?
For dev server:

1. Open .env.dev file.
2. Update REACT_APP_VERSION_NAME and REACT_APP_NAME with version name and number that you want to build.
3. Run npm run dev.

For staging server:

1. Open .env.staging file.
2. Update REACT_APP_VERSION_NAME and REACT_APP_NAME with version name and number that you want to build.
3. Run npm run build.
